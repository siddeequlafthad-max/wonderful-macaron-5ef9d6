# Siddiqul Afdad KM — Portfolio

Personal portfolio site for Siddiqul Afdad KM (Afthu), a data analyst based in Bengaluru, India.

**Live site:** https://wonderful-macaron-5ef9d6.netlify.app

## About

A single-page portfolio built around a dashboard-inspired visual language — report-style tabs, KPI tiles, and real Power BI screenshots — to match the subject matter: data analytics and BI dashboard work.

## Sections

- **Home** — intro and quick stats (listings analyzed, records segmented, tools, internship)
- **About** — background, interests, and photos
- **Case Studies** — two end-to-end projects with real dashboard screenshots:
  - Airbnb NYC Investment & Host Performance Analysis (Python, MySQL, Power BI, Power Query, DAX)
  - Customer & Marketing Analytics Dashboard (Power BI, DAX, SQL, RFM segmentation)
- **Services** — freelance Power BI dashboard work (Fiverr) plus data analytics, CRM/growth ops, and content creation
- **Contact** — email, LinkedIn, GitHub, Fiverr, resume download

## Tech

Single self-contained `index.html` — vanilla HTML, CSS, and JavaScript, no build step or dependencies. All images are embedded inline as base64 so the file works standalone. Deployed via [Netlify](https://netlify.com).
